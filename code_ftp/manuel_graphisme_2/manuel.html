Composition

Mise en page

Titre courant + notes de bas de page = inclus lorsque l'on arrange la surface de composition.

MAIS numéros de page + notes marginales =/= arrangement classique de la surface de composition.

Ouvrage = pages de droite + gauche = une paire = une unité graphique.

Surface de composition = disposée même façon partout.

Surfaces pas imprimées = fonction esthétique + repos pour l'œil.

Variation idéale de la proportionnalité = section dorée

Double page = bords extérieurs + marges extérieures plus larges que bords intérieurs + petit fond.

Blanc de fond = reflète visuellement quand double page ouverte + se voit doublé.

Largeur des bords extérieurs = +- somme de largeur bords intérieurs.

Folio = terme typographique pour numérode page + numéro de chapitre.

Titre courant = numéro de page / de chapitre + texte sur auteur / mots-clés / ...

Surface de composition = termes techniques suivants : grand fond + petit fond + bas de page + en-tête + titre courant + numéro de page + notes marginales.

En-têtes = informations brèves sur le contenu + structure le texte.

En-têtes = percutants + se démarquent du reste du texte.

Caractères gras / italiques = démarquer lees en-têtes.

Texte = divisé en paragraphes : rendre plus lisible + compréhensible.

Indentation = instrument important pour identifier nouveaux paragraphes + structurer le texte.

Premiers imprimés = place laissée début de paragraphe pour caractère alinéa H en rouge.

Indentation pas trop grande.

Veuve = dernière ligne paragraphe incomplète première ligne colonne.

Orpheline = première ligne paragraphe seule fin colonne + sur page créée par saut de page manuel.
